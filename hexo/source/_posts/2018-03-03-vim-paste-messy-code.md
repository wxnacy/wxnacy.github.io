Vim 让我头疼了很长时间的黏贴剪切版乱码的问题终于得到了解决，跟从 Vim 复制到剪切版是一样的道理，使用缓存区，妈蛋，为什么我没有早点想到。
<!-- more --><!-- toc -->
当复制系统内容后按键 `"+p` 即可将文本原格式黏贴到 Vim 中

方便的做法是使用 map 映射，修改 `~/.vimrc`
```vim
nnoremap <C-p> "+p
inoremap <C-p> <esc>"+pkddi
```
这样不管普通模式还是插入模式，使用 `<ctrl> + p` 既可以完成黏贴
