Python format 语句如何对大括号（{}）进行转义。

<!-- more -->

我想要达到一个效果，将

```python
'{:.2f}'.format(3.333)
```

中控制小数后位数的 `2` 进行参数控制，这样就要先对 `{:.2f}` format 一次，我这样写

```python
'{}:.{}f{}}'.format(2)
```

显然不对，我以前记录过如果想要显示 `{0}` 需要 &#123;&#123;0&#125;&#125;，而在这里，我想当然的以为将想要转义的部分单独用 `{}` 包起来就行了，愚昧。

真正的转义的方式

- &#123;&#123; 两个左卡括号输出 `{`
- &#125;&#125; 两个左卡括号输出 `}`

所以上面需求的正解为

```python
'{{:.{}f}}'.format(2)
```
