## 安装
### Pathogen
```bash
$ cd ~/.vim/bundle
$ git clone https://github.com/bronson/vim-trailing-whitespace
```

### Vundle
修改 `~/.vimrc`
```bash
Plugin 'bronson/vim-trailing-whitespace'
```
在 Vim 中运行
```bash
:PluginInstall
```

## 配置
```bash
map <leader><space> :FixWhitespace<cr>
```

## 使用
```bash
<leader><space>     # 即可去掉全部行为空格
```
