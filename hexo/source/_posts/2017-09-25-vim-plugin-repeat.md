## 安装
### Pathogen
```bash
$ cd ~/.vim/bundle
$ git clone https://github.com/tpope/vim-repeat
```

### Vundle
修改 `~/.vimrc`
```bash
Plugin 'tpope/vim-repeat'
```
在 Vim 中运行
```bash
:PluginInstall
```

## 使用
使用 `.` 即可重复操作，支持 [vim-surround](/2017/09/25/vim-plugin-surround/)
更多支持列表见 [文档](https://github.com/tpope/vim-repeat#repeatvim)
